## 📑 プロジェクト概要

MCPALLは、Model Context Protocol（MCP）に基づく多機能サービス集合プラットフォームで、様々なMCPサービスの迅速な開発と統合をサポートします。このプラットフォームは高度にモジュール化されており、新しいサービスモジュールを簡単に拡張できます。
